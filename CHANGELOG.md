# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog, and this project follows Semantic Versioning.

## [0.1.0] - 2026-06-11

### Added

- Initial command-line converter from Youzan Lakeboard `.lakeboard` files to XMind `.xmind` files.
- Optional XMind template mode for compatibility and styling.
- Python API via `convert_lakeboard_to_xmind`.
- Standard library test suite and GitHub Actions CI.
